--- title: Installing icon: fa-plug order: 2 ---

Install Roxana in your M2 repository:

{% highlight bash %} $ git clone https://github.com/rooting-company/roxana.git && ./roxana/mvnw.cmd install {% endhighlight %}

Obs: Infortunately you need to install Roxana manualy, because it is not in maven central repository yet. We are solving this problem. ; )

Add Roxana as dependency of your project:

For Maven users: {% highlight xml %} br.com.rooting roxana 0.0.5-SNAPSHOT {% endhighlight %} For Gradle users: {% highlight xml %} compile group: 'br.com.rooting', roxana: 'commons-lang3', version: '0.0.5-SNAPSHOT' {% endhighlight %}

Enable Roxana:

Just put @EnableRoxana in your Spring Rest API and that's it! {% highlight java %} @EnableRoxana @SpringBootApplication public class YourStringRestApplication { ... } {% endhighlight %}
